# Welcome Salon Management System

A high-end, full-stack web application designed for modern hair and beauty studios. This system combines a luxury frontend experience with a robust administrative backend.

## Key Features
- Premium Frontend: Dark-themed "Glassmorphism" UI with GSAP animations.
- Smart Appointments: Real-time scheduling with status tracking (Pending, Completed, Cancelled).
- Service Management: Fully dynamic services grid with premium Unsplash imagery.
- Admin Dashboard: Comprehensive analytics, customer management, and inbox for newsletter subscribers.
- Self-Healing Images: Dual-provider fallback system (Unsplash + Picsum) for guaranteed visual stability.
- User Management: Secure admin account management and credential updates.

## Tech Stack
- Backend: Laravel 10 (PHP 8.1+)
- Frontend: Blade, Bootstrap 5.3, jQuery, GSAP (Animations)
- Database: MySQL
- Icons & Fonts: FontAwesome 6, Google Fonts (Outfit)

## Installation
1. Clone the repository
2. Run `composer install`
3. Configure your `.env` file (Database & App Key)
4. Run `php artisan migrate --seed`
5. Start server: `php artisan serve --port=8888`
